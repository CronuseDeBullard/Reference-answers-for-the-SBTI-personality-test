# SBTI人格测试数据提取项目

## 项目说明

本项目从SBTI人格测试网站（https://sbti.unun.dev）提取了所有可能的测试结果和完整题库。

## 目录结构

```
SBTI人格测试/
├── README.md                    # 本说明文档
├── requirements.txt             # Python依赖

│   └── 网站源代码.html          # 测试网站的完整HTML源码
├── 02-提取工具/
│   └── 提取工具.py              # 从HTML中提取人格类型的Python脚本
├── 03-测试结果/

└── 04-废弃文件/                 # 开发过程中的临时文件
```

## 测试结果概览

### 人格类型总数
- 常规人格：25个
- 特殊人格：2个（DRUNK酒鬼、HHHH傻乐者）
- 总计：27个

### 测试结构
- 题目总数：30道常规题 + 2道特殊题
- 评估维度：15个维度，分为5大模型
- 计分方式：每题1-3分，每个维度2题

### 五大模型
1. 自我模型（S）：S1自尊自信、S2自我清晰度、S3核心价值
2. 情感模型（E）：E1依恋安全感、E2情感投入度、E3边界与依赖
3. 态度模型（A）：A1世界观倾向、A2规则与灵活度、A3人生意义感
4. 行动驱力模型（Ac）：Ac1动机导向、Ac2决策风格、Ac3执行模式
5. 社交模型（So）：So1社交主动性、So2人际边界感、So3表达与真实度

## 使用方法

### 1. 查看所有人格类型
打开 `03-测试结果/所有人格类型.json` 查看27个人格的完整描述。

### 2. 查看题库和答案
打开 `03-测试结果/完整题库与答案.md` 查看：
- 完整的30道题目
- 每个人格类型的标准答案（选项序号）
- 维度计分规则
- 匹配算法说明

### 3. 下载网站源代码
如果需要重新下载最新的网站源代码：
```bash
cd 02-提取工具
python 提取工具.py
```

提取工具会从 https://sbti.unun.dev 下载HTML源代码并保存到 `01-源代码/网站源代码.html`。

## 数据格式

### JSON格式（所有人格类型.json）
```json
{
  "人格代码": {
    "code": "人格代码",
    "cn": "中文名称",
    "intro": "简介",
    "desc": "详细描述"
  }
}
```

### 答案格式（完整题库与答案.md）
每个人格包含30题的选项序号（1/2/3），按照固定顺序排列。

## 匹配算法

1. 计算用户答案的15个维度得分
2. 将得分转换为等级（L/M/H）
3. 与25个标准人格模式计算距离
4. 选择距离最小的人格作为结果
5. 特殊判断：
   - 触发饮酒条件 → DRUNK（酒鬼）
   - 最高匹配度 < 60% → HHHH（傻乐者）

## 技术说明

### 数据获取方法
1. 使用Python的urllib库从网站下载HTML源码
2. 手动分析HTML源码中的JavaScript代码
3. 提取TYPE_LIBRARY常量中的所有人格类型定义
4. 整理题目和答案对应关系

### 优势
- 使用标准库，无需额外依赖
- 直接从官网获取最新数据
- 数据完整，包含所有字段
- 方法简单，易于理解和维护
